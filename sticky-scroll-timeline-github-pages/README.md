# Sticky Scroll Timeline

A small responsive timeline section for websites. When visitors scroll into the section, the panel sticks in place while the timeline progress bar moves downward and highlights each step.

## Files

- `index.html` - standalone demo page
- `styles.css` - scoped timeline styling
- `script.js` - scroll progress behavior
- `unlimited-elements-scroll-timeline-widget.md` - Unlimited Elements Pro widget setup

## WordPress / Elementor

For Elementor, use the Unlimited Elements Pro widget spec in `unlimited-elements-scroll-timeline-widget.md`.

If sticky positioning does not work in Elementor, check parent containers for `overflow: hidden`, `overflow: auto`, transforms, or motion effects.

## GitHub Pages

This folder can be published directly with GitHub Pages. Use `index.html` as the site entry point.
